CKD Risk Calculator

This App is a tool for health care professionists, patients and caregivers.

The aim is to evaluate the progression risk of Chronic Kidney Disease (CKD) based on the latest International Guidelines.

Reference

Kidney Disease: Improving Global Outcomes (KDIGO) CKD Work Group. KDIGO 2012 Clinical Practice Guideline for the Evaluation and Management of Chronic Kidney Disease. Kidney Int., Suppl. 2013; 3: 1-150.

When the albuminuria value is not available the calculator will provide only the eGFR value.

To estimate the Glomerular Filtration Rate (eGFR) we use the following formulas:

Cockcroft-Gault : 1,2

  1. SCr (mL/min) = (140-age) x weight/SCr x 72 (0.85 if female)

If patient's weight and height parameters are available the Calculator will automatically estimate the ideal weight (Keys formula) and the BSA value (DuBois formula) reporting GFR as a normalized value to body surface area.

If patient’s height parameter is not available, the Calculator will use the weight value reporting GFR as a normalized value to standard body surface area.

Keys formula for ideal weight:

Male = (height in m)² x 22,1 

Female = (height in m)² x 20,6

DuBois Formula (BSA):

 BSA = (W  0.425  x H  0.725 ) x 0.007184

where the weight is in kilograms and the height is in centimeters.

Reference:

DuBois D, DuBois EF. A formula to estimate the approximate surface area if height and weight be known. Arch Intern Medicine. 1916; 17:863-71.

MDRD - 4 parameters : 1,2

IDMS-traceable :

  1. GFR (mL/min/1,73 m 2 ) = 175 x (SCr) -1,154  x (age) -0,203  x 0.742 (if female) x 1.210 (if African American)

Not IDMS-traceable:

  1. GFR (mL/min/1,73 m 2 ) = 186 x (SCr) -1.154  x (age) -0.203  x 0.742 (if female) x 1.210 (if African American)

NOTE: The estimated GFR values above 60 mL/min/1.73 m2 should be interpreted as "above 60 mL/min/1.73 m 2 ", not an exact number.

CKD-EPI 2009 (single equation): 3  

  1. GFR = 141 x min(SCr/κ, 1) a  x max(SCr/κ, 1 )-1.209  x 0.993 age  x 1.018
    (if female) x 1.159 (if African American)

SCr: serum creatinin; MDRD- Modification of Diet in Renal Disease study; CKD-EPI: Chronic Kidney Disease Epidemiology Collaboration Equation ; k=0.7 (if female); k = 0.9 (if male); a = - 0.329 (if female); a = - 0.411 (if male); min: minimum of SCr/k or 1; max: maximum of SCr/k or 1

References:
